## Install

```
$ npm install --save cat-pad
```


## Usage

```js
const catPad = require('cat-pad');

catPad('Meow', 5);
//=> 'ğŸˆğŸˆğŸˆğŸˆğŸˆMeow'
```

## API

### catPad(input, count)

#### input

Type: `string`

String to pad.

#### count

Type: `number`<br>
Default: `0`

Amount of cats.
